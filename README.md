# PSN-Arduino

A lightweight Arduino implementation of the **PosiStageNet (PSN) v2.02** protocol.

## Features

- PSN v2.02 compliant
- Ethernet (W5500)
- Zero dynamic memory allocation
- Automatic INFO packets
- Automatic DATA packets
- Multiple trackers
- Quaternion support
- Euler angle conversion
- Arduino Library compatible

## Status

🚧 Under development

## Hardware

- Arduino Mega 2560
- W5500 Ethernet Shield
